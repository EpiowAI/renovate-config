# renovate-config

Org relay preset: new EpiowAI repositories onboard onto this automatically, which extends the Sylphx org-wide preset ([SylphxAI/renovate-config](https://github.com/SylphxAI/renovate-config)) — single source of truth per Doctrine ADR-6. Put EpiowAI-specific overrides in `default.json` here; base policy changes go upstream.
